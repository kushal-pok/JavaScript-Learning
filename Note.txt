 # Call Stack
 use LIFO(Last in first out)
 Eg:
    If there is three functions 
        1: One()
        2: Two()
        3: Three()

if we call function one() in first this will be execute first in global execution 
and out the function 

Anothere case :
If coder call one() function first and in function one() coder call another function Two()
and in function Two() coder call another function Three() too the LIFO(Last in first out) is 
implement in there first execute the function Three() and funtion Two() and function One()

Example code : 
functio one(){
    console.log("one")
    Two()
}

functio Two(){
    console.log("Two")
    Three()
}
function Three(){
    console.log("Three")
}
